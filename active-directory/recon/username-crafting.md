# Username crafting

Username crafting generates targeted username lists for AD enumeration and spraying from OSINT sources like LinkedIn names, following common patterns (e.g., first.last, flast, firstL). This fuels tools like `Kerbrute` for validation without creds.

* Username-Anarchy

```bash
./username-anarchy anna key
anna
annakey
anna.key
annakey
annak
a.key
akey
kanna
k.anna
...
```
